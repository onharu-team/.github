# 🌱 온하루
온하루는 나눔 카드를 소지한 아동이 주변 또는 원하는 지역의 나눔 가게를 쉽게 검색하고 예약할 수 있는 플랫폼입니다.

아동은 나눔 카드를 직접 제시하는 부담 없이 편리하게 나눔을 받을 수 있으며, 나눔 가게는 자연스럽게 선행을 실천하고 참여할 수 있습니다.
